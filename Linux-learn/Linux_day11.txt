程序：数据+指令
        程序风格：
                 过程式：以指令为中心，数据服务于指令
                 对象式：以数据为中心，指令服务于数据
         shell程序：提供了编程能力，解释执行
                          计算机运行二进制指令，为了方便人的理解，编程语言出现了。
                          编程语言：
                                低级：汇编语言
                                高级：
                                        编译执行：高级语言-->编译器-->目标代码 C  C++ Java
                                        解释执行：高级语言-->解释器-->机器代码 shell  perl  python
          过程式编程语言：     
                     顺序执行
                     循环执行
                     选择执行 
  
                     shell 过程式编程、解释执行
                           编程语言的基本结构：
                                              数据存储：变量、数组
                                              表达式
                                              语句
                            没有自带的内嵌函数（shell编程就是过程+Linux命令，shell程序是依赖系统上环境）
shell 脚本：文本文件
       执行脚本的解释器环境
       #!/bin/bash
       #!/usr/bin/python
       #!/usr/bin/perl
       shebang:
                    magic number：魔数
                   
执行简单的shell脚本
             bash first.sh 直接运行解释器，将脚本作为解释器参数运行
             ./first.sh  给予执行权限，使用文件路径指定文件执行

                    逻辑运算：true false
                       shell中0表示成功1表示true其他的表示false
                 
                       与
                       或
                       非
                       异或 不同为真相同为假
                       短路运算：
                                与：第一个为0，结果必定为0
                                       第一个为1，第二个必须参加运算
                                或：第一个为1，结果必定为1
                                       第一个为0，第二个必须参与运算
